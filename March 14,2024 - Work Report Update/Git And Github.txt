
1. Git is a free, open source version control tool that developers install locally on 
their personal computers, while GitHub is a pay-for-use online service built to run Git in the cloud.
2. Git Command
   a. git init
   b. git add -A
   c. git commit -m "index.html" ( file name )
   d. git checkout 
   e. git checkout development
   f. git push 
   g. git pull
   h. git log 
   k. git clone
